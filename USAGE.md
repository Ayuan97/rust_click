# 使用说明

## 这个软件是干什么的

这个工具不是录制器，而是一个参数编辑器。

它的作用是：

- 打开 `data/params/*.json` 这样的多武器参数文件
- 可视化每一发的 `x_steps / y_steps`
- 可视化 30 发累加后的理论轨迹
- 批量微调参数
- 直接拖动累计轨迹点来修正某一发
- 保存参数，并导出 HID Remapper 可导入的 JSON

## 先理解两张图

### 上图：每发步进

这张图看的是“第 N 发单独输出多少”。

- 蓝线：`X`
- 黄线：`Y`

适合看：

- 某一发是不是突然跳太大
- 某一段是不是抖动
- 中后段是不是不够平滑

### 下图：累计轨迹

这张图看的是“30 发一路累加后的整体轨迹”。

适合看：

- 整体有没有偏左 / 偏右
- 前几发是不是就开始跑偏
- 中后段是不是越来越歪

下图里的点可以直接拖。

## 推荐的调参顺序

1. 左侧选一把武器。
2. 点一次“设为基准”。
3. 先调 `Y`，把纵向节奏调顺。
4. 再调 `X`，处理左右偏移。
5. 先调前 5 到 10 发。
6. 再调中段和后段。
7. 最后保存并导出。

## 怎么改参数

### 1. 直接改表格

右侧 `30 发参数表` 可以直接改：

- `X`：左右补偿
- `Y`：纵向补偿

适合做精确修改。

### 2. 批量按钮

右侧有这些按钮：

- `X +1 / X -1`
- `Y +1 / Y -1`
- `平滑 X / 平滑 Y`
- `缩放 X / 缩放 Y`

规则：

- 如果你先在表格里选中了几行，就只改那几发
- 如果没有选中任何行，就会默认作用于全部 30 发

建议：

- 小改：用 `+1 / -1`
- 大改：用 `缩放`
- 曲线太乱：用 `平滑`

### 3. 直接拖累计轨迹点

这是最好用的功能之一。

在下方 `累计轨迹` 图里：

- 拖第 `N` 发的点
- 软件会自动把这次位移换算成第 `N` 发的 `X / Y step`
- 后续累计轨迹会一起变化

适合：

- 局部修第几发
- 快速看“如果这发再往左一点 / 往下一点会怎样”

## 基准对比怎么用

工具栏里有：

- `设为基准`
- `清除基准`

设为基准以后：

- 虚线：基准
- 实线：当前

这对“调了一圈以后到底变好了还是变坏了”非常有用。

推荐用法：

1. 先打开一份你现在能用的参数
2. 点 `设为基准`
3. 再开始改
4. 随时看当前曲线和基准差异

## 撤销 / 重做

支持这些操作的撤销 / 重做：

- 改表格
- 批量加减
- 平滑
- 缩放
- 改武器参数
- 改全局倍率
- 拖轨迹点

如果改坏了，先别慌，直接 `撤销`。

## 左侧几个参数怎么理解

### 全局预览

- `基础倍率`
- `FOV 补偿`
- `X 微调倍率`
- `Y 微调倍率`

这些会影响当前预览出来的理论轨迹。

一般建议：

- 先尽量少改全局参数
- 优先把单武器的 `x_steps / y_steps` 调顺

### 武器参数

- `射击间隔`
- `起始延迟`
- `攻击窗口`
- `1倍镜倍率`
- `8倍镜倍率`
- `站立倍率`
- `蹲下倍率`

如果你的问题明显是“节奏不对”，就先看 `射击间隔 / 起始延迟`。
如果问题更像“轨迹形状不对”，就先改 `x_steps / y_steps`。

## 一套实用打法

如果你完全不知道从哪里开始，按这个顺序来：

1. 打开参数文件
2. 选武器
3. 点 `设为基准`
4. 只盯前 10 发
5. 先用 `Y +1 / Y -1` 调纵向
6. 再用 `X +1 / X -1` 调左右
7. 某几发不好修时，直接拖下方轨迹点
8. 某段太毛躁时，用 `平滑`
9. 整段偏大或偏小时，用 `缩放`
10. 满意后保存并导出

## 导出

调完以后：

1. 点 `保存`
2. 点 `导出 JSON`
3. 选输出路径
4. 把生成的 JSON 导入 HID Remapper

## 当前限制

当前 GUI 主要支持：

- 多武器参数格式：`global + weapons[]`

暂不支持：

- 单武器 `ak_tune_params` 那套格式的完整可视化编辑
